Career Guidance Tool

Career guidance Tools help individuals acquire the knowledge, skills, and experience necessary to identify options, explore alternatives and succeed in society. This application is designed to assist individuals in making and implementing informed educational and occupational choices.

This app allows user to give its inputs like Course of Study, Number of colleges to display, Type of degree, State, Region, College Cost, & Earning where output is shown in both in a geographical map format and a table output, where the Colleges are Ranked based upon the choices made. Additionally, data can be visualized as a bar graph to compare the costs between different institutions.

Instructions for navigating this app:

  • User is asked for inputs and can choose from:

  • Dropdown list: To Choose the course of study

  • Slidernoption to choose for Number of colleges to Display

  • Radio buttons to Choose rank by

  • Checkbox for Degree(s) to rank(display)

  • Dropdown to choose State, Region

  • Slider option to Choose Tuition Range

  • Slider option to Choose Earning Range

  • US geographical map shows the college names corresponding to input, showing both states and region wise data.

  • When the user hover over a selection, the marker in geographical map changes it color to highlight the selection.

  • Another tab represents the cost comparison between the different institutions.

  • All Code is written in R.
